# Path Optimization using Ant Colony Optimization (ACO)

## Overview
This project implements the Ant Colony Optimization (ACO) algorithm to find optimal paths between nodes considering distance and traffic factors.

## Features
- Traffic-aware path optimization
- Pheromone-based route selection
- Route visualization
- Cost calculation using distance and traffic

## Technologies Used
- Python
- NumPy
- Matplotlib

## How to Run
1. Install dependencies
2. Run:
   python main.py
## Alternative Method
Just Copy the code in Jupyter notebook and run.
